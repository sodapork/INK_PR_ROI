PR ROI Calculator Widget

A lightweight, embeddable tool to estimate marketing ROI

How to Embed

Add this code to your website:

<!-- 1. Add the script --> <script src="pr-roi-calculator.js"></script> <!-- 2. Add a container div --> <div id="pr-roi-calculator"></div> <!-- 3. Initialize the calculator --> <script> new PRROICalculator('pr-roi-calculator'); </script>

Customization Options

// Light theme (default)
new PRROICalculator('pr-roi-calculator', {
    theme: 'light',
    currency: '$'
});
